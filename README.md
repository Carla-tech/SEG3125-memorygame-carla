# GlowMatch Memory Game - SEG3125 Assignment 3

Designer: Carla El Hajj Ali  
Course: SEG3125 - Analysis and Design of User Interfaces

## Project description
GlowMatch is a high-fidelity interactive memory game prototype. Users choose a difficulty level and a visual theme, then practice short-term visual memory by flipping cards and matching hidden pairs.

## Netlify build settings
- Branch: `main`
- Base directory: leave empty
- Build command: `npm run build`
- Publish directory: `dist`

## Technical notes
This project is built with React, Vite, Bootstrap, HTML, CSS, and JavaScript. It includes:
- `.nvmrc` to force Node 20 on Netlify
- `.npmrc` to force the public npm registry
- `netlify.toml` for build settings and redirects
